### About the Experiment 

This is a virtual simulation focused on Heart Rate Variability (HRV) Analysis. This lab transitions from basic heart rate monitoring to the sophisticated study of the variation in time intervals between consecutive heartbeats, known as R-R intervals. The simulation allows users to process Electrocardiogram (ECG) data to extract these intervals and perform both time-domain and frequency-domain analyses. Students can explore how the Autonomic Nervous System (ANS)—specifically the balance between the sympathetic and parasympathetic branches—influences heart rate fluctuations. By adjusting physiological states or observing different data sets, users learn to calculate key HRV metrics such as SDNN (Standard Deviation of NN intervals) and the LF/HF ratio, which are vital clinical indicators of stress, cardiovascular health, and recovery. This virtual module provides a critical platform for understanding advanced signal processing and physiological feedback loops without requiring high-end clinical ECG hardware.
